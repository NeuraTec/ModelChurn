# 📊 Churn Prediction Project (End-to-End ML System)

Este proyecto implementa un sistema completo de predicción de **churn de clientes**, desde el entrenamiento del modelo hasta su despliegue como API y aplicación web interactiva.

El objetivo es demostrar cómo un modelo de Machine Learning puede integrarse en un sistema real tipo producto.

---

## 🚀 Demo en vivo

- 🖥️ Web App (Streamlit):  
  👉 https://tu-app.streamlit.app

- ⚙️ API (Flask en Render):  
  👉 https://tu-api.onrender.com

---

## 🧠 Problema

El churn (abandono de clientes) es un problema crítico en marketing y empresas de servicios.  
Este proyecto busca predecir la probabilidad de que un cliente abandone un servicio, permitiendo tomar acciones preventivas.

---

## 🏗️ Arquitectura del sistema

- `ml_churn_prediction/` → Entrenamiento del modelo y pipeline de ML  
- `churn_api/` → API en Flask (deploy en Render)  
- `churn_app/` → Interfaz web en Streamlit  

### 🔄 Flujo general:
Usuario → Streamlit App → API Flask → Modelo ML → Predicción
